Some of my past projects

ARMIRA

This project is a web-based system designed to streamline and digitize the management of assets and the approval process of asset requests within the BKKBN. The system provides a centralized platform for tracking physical assets, handling asset request submissions, and managing approval workflows efficiently and transparently.

Dashboard Mondar Mandir

This project is a web-based dashboard built with the PRM Division of Telkom Indonesia. It helps monitor document workflows and manage completed documents in one place. Users can track the status of documents in real-time—such as drafting, reviewing, approval, and archiving—and access finalized files securely through an organized directory.

Gasboard

This project is a comprehensive data integration and visualization developed for PT Pertamina Gas. It focuses on monitoring key aspects of gas and oil distribution operations, including revenue tracking, safety performance, and asset integrity management.

Giselle

This project is a guest list and document collection system developed for PT Telkom Indonesia to track and manage the circulation of documents among Telkom partners. It ensures visibility, confirmation, and accountability at each stage of the document handover process.

MyMonalisa

This web-based system was developed to help PT Multimedia Cakra Indonesia monitor and manage the renewal of vehicle documents for company assets. It tracks document validity (e.g., STNK, tax), sends timely reminders. The system ensures compliance, minimizes renewal delays, and improves vehicle document management efficiency.

Klikbengkel

This project is a web-based system developed for PT Multimedia Cakra Indonesia to manage and track the maintenance of company-owned motor vehicles. The system allows for scheduled maintenance tracking, service history recording, and the management of repair tasks to ensure that all vehicles are in optimal condition.
